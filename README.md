# CloudVAPT

Cloud Vulnerability Assessment and Penetration Testing Tools

## Features
- Automated cloud security scanning
- IAM privilege escalation detection
- Multi-cloud compliance auditing
- Kubernetes security assessment
- Container security scanning

## Tools
- IAM Scanner
- S3 Bucket Scanner  
- Network Security Scanner
- CIS Compliance Scanner
- Kubernetes RBAC Auditor

## Usage
Each tool can be run independently for cloud security assessment.
